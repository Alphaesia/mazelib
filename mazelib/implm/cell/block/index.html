Module mazelib::implm::cell::block

source ·
Expand description

Block cells are cells where each cell is exactly one of: passage, wall, boundary, or unvisited.

They are named block cells because they have a blocky or pixellated appearance.

    CellLocation for block cells.
  • BlockCellValue
    A cell type where cells are either passage cells or wall wall cells, with no in between. They are called Block Cells because the resulting mazes look blocky / pixellated.
