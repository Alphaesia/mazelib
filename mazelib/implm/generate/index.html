source ·

A variety of different maze generators built-in.

The documentation for each algorithm contains a brief overview of the algorithm, the texture it produces, and some notable performance characteristics to be aware of.

ThinkLabyrinth has a page that compares all the algorithms featured here and provides more detail and examples.

The Hunt-and-Kill algorithm is recommended as the default algorithm. It produces high-quality mazes while maintaining high performance.
