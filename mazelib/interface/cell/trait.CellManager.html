Expand description

Translates between points, cell locations, and cell identifiers. Executes queries on mazes and modifications to mazes.

Cells have no inherent meaning. They must be interpreted in context. This is exactly what a Cell Manager does.

Due to requiring intimate knowledge of how the coordinate space works and how the cell space works, each Cell Manager is specific to a single CoordinateSpace-CellValue pair. There is generally only one such manager for each pair.

    from: <<Self as CellManager>::CoordSpace as CoordinateSpace>::PtType,
    to: <<Self as CellManager>::CoordSpace as CoordinateSpace>::PtType
) -> ConnectionType

Return the type of connection (graph theory: edge) between two points.

If you attempted to walk from from to to this is what you would encounter.

The order of the arguments is important and has semantic meaning. Swapping the arguments may produce different results.

This method follows the priority order outlined in ConnectionType.
