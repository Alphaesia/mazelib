Expand description

Value of a cell.

Cells can be marked. Marks are a general-purpose signalling mechanism, and have no universal interpretation. It is recommended you clean up any marks you make, as other components may make marks of their own and get confused by pre-existing marks.

fn is_fully_visited(&self) -> bool

If the cell has not been fully generated (visited). Partially-generated cells are not considered fully visited. This is because when applying templates, such as a solid boundary around the edge of the maze, the cells that get touched should still be considered by generators.
